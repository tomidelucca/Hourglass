# Hourglass

## How to build

To build Hourglass, run the following commands:

```bash
git clone https://github.com/tomidelucca/hourglass.git
cd hourglass
mvn clean package
java -jar target/hourglass-1.0-SNAPSHOT.jar 
```

## How to run

```bash
java -jar target/hourglass-1.0-SNAPSHOT.jar [parameters]
```
